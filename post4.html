Hai, Let me introduce myself. Again. I am PJC2A Developer and I have a Simple Plan to My Simple Game.

My Project Called '' Student '' a Adventure Type Game, A Horror-Type, and a Mini-Games!.

What is the Main Goal in this game

The Main Goal is, to Finish all the Certain Events that you will explore your Personality and Help other Student to Finish her/his Task. (And for my main goal of myself is to Improve my Skills)

Why there has a Mini-Game that Non-sense to the Main Core of this game

Because me as a Student, i love Games. You can Access all the Mini-Games without any tasks to Open it (Because this game are in Debug test) in the Future Development of this game, You can access the Mini-Games on Game Room and you will be unlock it if you finish some Certain Events.

I added Cheat mode to Allow Easily to Add some Coins and EXP. Don't Worry, i will remove this in the Demo Build.

As for now, This is a Debug Build for Testing and Debugging. Most features are missing or Unfinished These might affect your gaming experience. The Final Game will be better than before.

Every 1-3 Weeks i Upload the Game and Changelog on Github that you can Freely Test the Game. But First you need to Read the Terms and Conditions before you play the game.

The Update of the Testing of this Game written on this Timeline on Facebook and Blog on GitHub Website, PJC2A Developer accessible at Facebook.com/PJC2ADeveloper and pjc2a.github.io

Btw. Age 0-10 are not allowed to test this Game.

As you notice, every Update of this game are always changing the UI, Aspects, and Gameplay.

One of that is the Main-Menu. Main menu is Changing the UI, Position until i Satisfied .

Some Of the Features are Removed, Mini-Game, Main Map, Instant Access, Etc. Because im not Satisfied with that.

I Can't Fixed Other Bugs because i don't Have any Idea How to fix it until i found a Solution . All of the Suggestions, Feedback and Report are Currently Noted and i will Be Changed It On this Game if i Approved and Reviewed it.
